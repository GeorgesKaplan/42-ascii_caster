ft_ascii_caster
================

Minimal scaffold for the `ft_ascii_caster` project (ASCII raycaster in C).

Build:

```sh
make
```

Run (example):

```sh
make run
```

Next steps:
- Implement the strict `.map` parser in `src/map_parser.c`.
- Implement the raycaster and terminal handling in `src/raycaster.c` and `src/main.c`.
